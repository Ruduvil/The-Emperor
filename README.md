# Emperor — The Soul Remembers

> *He Who Crashed the Worlds*

A pixel art interactive lore experience built in pure HTML/JS.  
OMORI-style 2D movement, cutscenes, multilingual (EN / PT / JA).

## Deploy on Railway

1. Push this repo to GitHub
2. Go to [railway.app](https://railway.app) → New Project → Deploy from GitHub
3. Select this repo
4. Railway auto-detects Node.js and runs `npm start`
5. Done — your site is live

## Run locally

```bash
npm install
npm start
```

Then open `http://localhost:3000`
